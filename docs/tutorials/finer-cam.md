# Finer-CAM

> Better explanation for classification decisions on similar species.

## Learning Objectives

By the end of this tutorial, you will be able to:

1. Understand how Finer-CAM extracts the parts of an organism image that are most useful for telling one similar species from another.
2. Use the provided tutorial or demo to create visual explanation maps for a species image.
3. Judge whether the highlighted regions match biologically meaningful traits provided by ecologists or field experts.

## Prerequisites

- We provide demos and a tutorial notebook to support hands-on use, so there will be no need for setting up local environments.
- If local environments are preferred, you can install the package by `pip install grad-cam`.

## Setup

- Prepare organismal images of interest with corresponding species labels and upload them to Google Drive, so that they can be used in Colab to train models and produce explanation. A training-test split of images would be preferred.
- If you want to run the method locally, you can train a classifier with the prepared images. We use a linear classifier connected to a DINOv2 backbone in our example.
- You can also use the datasets and models prepared by us, following the instructions at [Finer-CAM repo](https://github.com/Imageomics/Finer-CAM).

## Background

Finer-CAM is designed for explaining model decisions in situations where the species of interest look very similar. 
Standard explanation tools tend to highlight generic organism regions instead of the subtle trait that truly separates similar species.
Finer-CAM identifies discriminative traits by asking directly, what makes the species different from a similar one.

## Steps

1. Train a neural network with prepared images.
2. Select an image of the organism you want to inspect. Choose a case where the distinction is subtle, such as two closely related species.
3. Run Finer-CAM and Grad-CAM through the provided notebook to generate visual explanation maps. 
4. Inspect the highlighted regions and ask whether they align with known field traits, such as wing color, markings, shape, or other subtle morphology. Compare the regions generated by Finer-CAM and Grad-CAM and check whether the results of Finer-CAM reflect more discriminative traits.
5. Use quantitative metrics to evaluate the effectiveness of Finer-CAM.
6. Additionally, we show how Finer-CAM can be used to extract traits with natural language.

## Summary

Finer-CAM is a visual explanation tool for fine-grained recognition. It helps reveal which visible traits a model is using to separate similar species.
The workflow is lightweight and straightforward. Once there is a model, Finer-CAM will be a low-latent tool to interpret and check whether the model's reasoning matches biological intuition.
